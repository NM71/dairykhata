# Dairy Khata

A Customized Flutter App for Milk Sellers (Mostly in Villages)

It includes:
- Splash Screen
- CRUD Operations
- Record Keeping
- Reciept Generation based upon timeline
- Local Storage
- Customized Milk Rates
